The Idea

Immersive theater meets game design. A fully scripted branching show performed live, where actors move seamlessly based on audience input with zero improvisation.

Branching stories are easy to pitch. They’re hard to execute live.

Design Problems We Had To Solve

Narrative integrity

  • No backward writing
  • Each split multiplies risk
  • Choices must matter

Choice transmission

  • Actors not in the room
  • Unambiguous scene cues
  • Zero room for error

Seamless flow

  • Entrances and resets
  • Close range realism
  • Hard time window

How We Implemented Choices

  • Audience makes the decision in room
  • Decision triggers a cue light state
  • Offstage actors receive the state instantly
  • Next scene begins with no pause
Physical cue-light system and scene entrance sheets backstage.

Season 1 Debrief

Held up

  • Story Integrity
  • Direction
  • Performances
  • System Reliability

Primary gap

  • Immersion consistency

    • Believability varied by room context and ticketed framing.

Why

  • Theater corner vibe
  • Space read as "set"
  • Context fought belief

What Season 2 Fixes

Season 2 is immersion first. We stage contained scripted moments in real places so the moment feels like it is actually happening. The location is the set, and the context supports believability instead of fighting it.

  • Location chosen for realism, not convenience
  • Contained "moment zone" staging
  • Entry and exit beats designed to avoid "show framing"
  • Audience perspective designed to feel like witnessing

Season 1 proved the system. Season 2 proves the premise.
